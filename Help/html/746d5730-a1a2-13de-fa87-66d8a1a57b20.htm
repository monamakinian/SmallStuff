RingBufferT Class
A RingBuffer is a circular list with a fixed capacity. Adding more items to the RingBuffer than its capacity will overwrite the the oldest item.

Examples
The following example adds 10 ints to a RingBuffer with a capacity for 5. The it lists the contents from oldest to newest:
Copy
var rb = new RingBuffer<int>(5);
for(int i=0; i<10; i++){ rb.Add(i); }
foreach(var item in rb){ Console.WriteLine(item); }

ExtraContainers Namespace
All items in a RingBuffer can be iterated over from oldest to newest by using the enumerator returned by the GetEnumerator method, which allows for the standard foreach syntax.
It is also possible to iterate over the items from newest to oldest with the enumerator returned by the GetReverseEnumerator method.
